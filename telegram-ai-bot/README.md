# Telegram AI Bot

Бот для Telegram с интеграцией LLM через OpenRouter API.

## Требования

- Python 3.11+
- Docker (опционально)
- Telegram Bot Token
- OpenRouter API Key

## Установка

### Локальный запуск:

1. Клонируйте репозиторий
2. Скопируйте `.env.example` в `.env`
3. Заполните `.env` своими ключами
4. Установите зависимости:
   ```bash
   pip install -r requirements.txt
